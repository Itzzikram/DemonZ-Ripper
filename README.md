# DemonZ-Ripper
Rip 3D models from Fab.com &amp; WebGL sites. Extracts meshes, textures, shaders, and materials via WebGL2 API interception. Exports to GLB, glTF 2.0, OBJ, and Unreal Engine UAsset. Features React UI with real-time progress, Fastify + Puppeteer backend, PBR material pipeline, and Docker/K8s/Terraform deployment support.
